# JOVI Modo Aula - Camera Inteligente para Estudantes

## Challenge FIAP x JOVI 2026 - Sprint 2

### Sobre o Projeto
O projeto **JOVI Modo Aula** apresenta uma proposta de interface para uma funcionalidade academica de camera inteligente, focada em organizar registros por materia, facilitar a captura de conteudo de aula e demonstrar recursos de apoio ao estudo.

### Entrega Principal da Sprint
Esta entrega principal foi desenvolvida com:

- HTML5 semantico
- CSS3 com Flexbox
- JavaScript puro, sem frameworks no front principal

Paginas entregues:

- `index.html` - pagina inicial
- `sobre.html` - contexto do projeto e equipe
- `funcionalidades.html` - funcionalidades detalhadas e slideshow
- `prototipo.html` - demonstracao navegavel do fluxo do app
- `contato.html` - login demonstrativo e formulario de contato

Arquivos de apoio:

- `css/style.css` - estilos responsivos e layout com Flexbox
- `js/slideshow.js` - slideshow com manipulacao de eventos
- `js/validacao.js` - validacao de formularios, login e eventos DOM
- `js/modo-aula.js` - prototipo interativo com persistencia local no navegador
- `INTEGRANTES.TXT` - nome completo e RM dos integrantes

### Requisitos Atendidos

- Layout dinamico e moderno para navegacao entre as paginas
- Estruturacao semantica com `header`, `nav`, `main`, `section`, `article` e `footer`
- Uso de Flexbox para posicionamento e organizacao dos conteudos
- Manipulacao dinamica do DOM com JavaScript
- Validacao de formularios e login demonstrativo
- Uso de alertas, prompts e eventos de usuario
- Manipulacao de imagens com slideshow
- Projeto principal feito sem frameworks no front-end
